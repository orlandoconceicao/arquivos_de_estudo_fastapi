# 🚀 API REST com FastAPI

Projeto backend desenvolvido para **praticar a construção de APIs REST modernas com FastAPI**, aplicando boas práticas de organização, validação de dados e documentação automática.

---

## 🎯 Objetivo do projeto

Este projeto tem como objetivo consolidar conhecimentos em:

- Desenvolvimento de APIs REST com **FastAPI**
- Criação de endpoints (`GET`, `POST`, `PUT`, `DELETE`)
- Validação e tipagem de dados com **Pydantic**
- Estruturação de projetos backend em Python
- Uso de documentação automática (Swagger / Redoc)

---

## 🧠 Decisões técnicas

- **FastAPI** foi escolhido pela alta performance, simplicidade e tipagem nativa
- **Pydantic** utilizado para garantir validação e consistência dos dados
- Separação de responsabilidades (rotas, schemas, modelos) para facilitar manutenção e escalabilidade
- Estrutura preparada para crescimento do projeto

---

## 🛠️ Tecnologias utilizadas

- Python 3.10+
- FastAPI
- Uvicorn
- Pydantic
